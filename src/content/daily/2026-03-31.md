# ☀️ AI 晨报 — 2026年3月31日 周二

## 🔥 跟你直接相关

### 微软大更新：Copilot Cowork 上线 + Researcher 用 Claude 审 GPT 的活
Reuters / GeekWire / 微软官方博客 — 微软昨天一口气放了三个大招：① **Copilot Cowork** 进入 Frontier 早期体验，可以跨 M365 应用执行长链条多步骤任务，不用你一直盯着；② Researcher 新增 **Critique 功能**——GPT 先写初稿，Claude 再审准确性、完整性和引用质量，双模型协作；③ 新增 **Council** 功能，并排对比不同模型输出。Critique 在 Deep Research 评测上比 Perplexity（Claude Opus 4.6）高 7 分。
**→ 你在微软做 AI Search，这三个都是自家产品。GPT+Claude 协作是正式承认"没有一个模型能全包"，这个架构思路对你设计 AI 体验有直接参考。Cowork 的"描述结果→自动规划→可视进度→随时插手"也是 agent UX 的标杆。**

### Claude 付费用户今年翻倍，但 ChatGPT 仍遥遥领先
TechCrunch / PYMNTS — 2800 万美国消费者支付数据显示，Claude 付费订阅 2026 年翻了一倍以上，1-2 月新增订阅创历史记录。驱动力：超级碗广告、Computer Use 和 Claude Code 发布、五角大楼事件带来的用户支持。但 ChatGPT 每周仍吸引 13,500-21,000 新订阅者，Claude 的差距还很大。
**→ 你每天用 Claude，它在赢但没赢够。结合上周市场份额从 29% 跌到 13% 的数据，Anthropic 的消费端在涨、API 端在被中国模型蚕食，两条线走势相反。**
