# ☀️ AI 晨报 — 2026年4月1日 周三

## 🔥 跟你直接相关

### Anthropic 把 Claude Code 源码泄露了——51.2 万行 TypeScript 全裸奔
Fortune / The Register / VentureBeat / CyberNews — 安全研究员 Chaofan Shou 发现 Claude Code v2.1.88 的 npm 包里带了一个 59.8MB 的 source map 文件（cli.js.map），包含完整未混淆的 TypeScript 源码。这是 Anthropic 几天内第二次安全失误（上次是意外暴露 Mythos 内部项目名）。源码里还藏了个 "Undercover Mode" 系统，专门设计来防止内部信息泄露——讽刺的是，这个防泄露系统本身也泄露了。Anthropic 回应称"是打包人为失误，不是安全漏洞，无用户数据泄露"。社区已把源码镜像到 GitHub，有人开始用 Rust 重写。
**→ 你每天用 Claude Code。技术上不影响使用（已修复到 2.1.89），但这暴露了 agent 工具链的供应链安全问题——npm 生态的老毛病，现在 AI 工具也中招了。更有意思的是源码分析揭示了 Claude Code 的架构设计，对理解 agent CLI 怎么做很有参考。**
