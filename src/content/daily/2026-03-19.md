### Claude Code Skill 自进化：56% → 92%，4 轮迭代
@Hesamation ❤️1385 — 受 Karpathy autoresearch 启发，有人写了一个 skill，专门自动 fine-tune 其他 Claude Code skill。运行 → 评估 → 微调指令 → 再运行 → 保留有效调整。一个 skill 从 56% 准确率跑到 92%，只花了 4 轮。
