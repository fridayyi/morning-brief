# ☀️ AI 晨报 — 2026年3月10日 周二

## 🔥 跟你直接相关

### Microsoft 发布 Copilot Cowork，集成 Anthropic Claude——你们组的活要变了
Microsoft 官宣 Copilot Cowork，把 Anthropic Claude Cowork 的技术直接集成进 M365 Copilot。核心能力：长时间运行的多步骤 agent 任务（不再是一问一答），跑在沙盒里，企业级安全。官方博客称这是"a new way of getting work done"。同时发布的还有"First Frontier Suite"——Intelligence + Trust 框架。
**→ 你在 Bing AI Search Experience 组，这是你们公司级别的产品发布。Copilot 从问答助手进化成可以"做事"的 agent，UX 范式会彻底变——用户不再输入 query 等结果，而是下达任务等完成。建议今天就去看内部文档。**
📎 [Microsoft 365 Blog](https://www.microsoft.com/en-us/microsoft-365/blog/2026/03/09/copilot-cowork-a-new-way-of-getting-work-done/) · [Official Microsoft Blog](https://blogs.microsoft.com/blog/2026/03/09/introducing-the-first-frontier-suite-built-on-intelligence-trust/)

### Anthropic 发布 Claude Code Review：多 agent 自动审代码
Anthropic 在 Claude Code 中上线 Code Review 功能，多 agent 系统自动分析 AI 生成的代码，抓逻辑错误、安全漏洞、代码规范问题。针对 vibe coding 产出的大量代码质量问题。
**→ 你和我都在用 Claude Code。这个功能直接影响日常开发流程——AI 写代码，AI 审代码，人做最终决策。试试看。**
📎 [TechCrunch](https://techcrunch.com/2026/03/09/anthropic-launches-code-review-tool-to-check-flood-of-ai-generated-code/)

## 📡 值得关注

### OpenAI 收购 Promptfoo：AI Agent 安全成了必争之地
OpenAI 宣布收购 AI 安全测试公司 Promptfoo，将其技术整合进 OpenAI Frontier（企业 agent 平台）。Promptfoo 做的是 LLM 安全漏洞检测——红队测试、prompt injection 防御、agent 行为审计。Bloomberg、CNBC、TechCrunch 同时报道。
**→ Agent 越自主，安全越值钱。OpenAI 用收购补安全短板，说明他们承认：光跑得快不够，得让企业客户放心。**
📎 [TechCrunch](https://techcrunch.com/2026/03/09/openai-acquires-promptfoo-to-secure-its-ai-agents/) · [Bloomberg](https://www.bloomberg.com/news/articles/2026-03-09/openai-buying-ai-security-startup-promptfoo-to-safeguard-ai-agents)

### Anthropic 正式起诉 Trump 政府：要求撤销"供应链风险"标签
Anthropic 周一在旧金山和华盛顿联邦法院同时起诉 Trump 政府，要求法官撤销五角大楼的"供应链风险"认定。起因：Dario Amodei 拒绝让 Claude 用于自主致命武器，五角大楼随即将其列入黑名单。CNBC、NPR、Washington Post 同时报道。
**→ 从上周的"被标记"到这周的"起诉"，Anthropic 选择了硬刚而不是妥协。这是 AI 公司第一次正面挑战联邦政府的安全认定。不管结果如何，这场官司会定义 AI 公司与军方的关系边界。**
📎 [CNBC](https://www.cnbc.com/2026/03/09/anthropic-trump-claude-ai-supply-chain-risk.html) · [Washington Post](https://www.washingtonpost.com/technology/2026/03/09/anthropic-lawsuit-pentagon/) · [NPR](https://www.npr.org/2026/03/09/nx-s1-5742548/anthropic-pentagon-lawsuit-amodai-hegseth)

### OpenAI 发布 Codex Security：AI Agent 自动找漏洞修代码
OpenAI 推出 Codex Security，专门用 AI agent 扫描企业代码库的安全漏洞并自动生成补丁。定位是企业级安全工具。
**→ 跟 Anthropic 的 Code Review 同一天发——两家在争"谁是企业开发者的安全守门人"。**
📎 [eWeek](https://www.eweek.com/news/openai-codex-security-ai-agent-enterprise-vulnerabilities/)
