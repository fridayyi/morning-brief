# ☀️ AI 晨报 — 2026年3月9日 周一

## 🔥 跟你直接相关

### OpenAI 机器人负责人 Caitlin Kalinowski 辞职：Pentagon 交易没有护栏
OpenAI 硬件/机器人团队负责人 Caitlin Kalinowski 周六公开辞职，原因是 OpenAI 与五角大楼签署的机密网络部署协议"没有定义足够的 guardrails"。她在 X 上写道："对美国人的大规模监控和没有人类授权的致命自主权，值得比现在更多的审慎。"Reuters、Bloomberg、TechCrunch、NPR、Fortune 同时报道，这是 Pentagon 交易引发的最高级别人才流失。
**→ OpenAI 内部分裂在加速。Kalinowski 是前 Meta 硬件高管，不是边缘人物。如果更多核心员工跟进，OpenAI 的机器人业务会受实质影响。对 M365 生态的直接影响不大，但品牌风险在累积。**
📎 [Reuters](https://www.reuters.com/business/openai-robotics-head-resigns-after-deal-with-pentagon-2026-03-07/) · [TechCrunch](https://techcrunch.com/2026/03/07/openai-robotics-lead-caitlin-kalinowski-quits-in-response-to-pentagon-deal/) · [Fortune](https://fortune.com/2026/03/07/openai-robotics-leader-caitlin-kalinowski-resignation-pentagon-surveillance-autonomous-weapons-anthropic/)

### Cursor 发布 Automations：AI Agent 从编辑器走向 DevOps
Cursor 推出 Automations 功能——AI agent 不再只在编辑器里写代码，现在可以被 Slack 消息、GitHub PR、PagerDuty 告警、定时器自动触发。自动 code review、自动 PR 批准、自动 incident response，不需要手动输入 prompt。Cursor 内部已经在用。
**→ 这就是 vibe coding 的下一步：从"人驱动 AI 写代码"到"事件驱动 AI 处理整个 DevOps 流程"。OpenClaw 的 cron + heartbeat 做的是类似的事，但 Cursor 把它产品化了。值得体验。**
📎 [Dev Weekly](https://singhajit.com/dev-weekly/2026/mar-2-8/gpt-54-cursor-automations-us-agencies-anthropic-oracle/)

## 📡 值得关注

### Nvidia 宣布不再追加投资 OpenAI 和 Anthropic：IPO 要来了
Jensen Huang 在 Morgan Stanley 技术大会上明确表示，Nvidia 对 OpenAI 和 Anthropic 的最近投资"很可能是最后一笔"。原因：两家公司今年都将 IPO，届时 Nvidia 可以在公开市场交易，不需要继续当 VC。
**→ OpenAI 和 Anthropic 的 IPO 从传言变成了 Nvidia CEO 亲口确认的事。这对整个 AI 生态的资本结构是分水岭——从私募融资转向公开市场定价。**
📎 [Bloomberg](https://www.bloomberg.com/news/articles/2026-03-07/openai-s-head-of-robotics-resigns-over-company-s-pentagon-deal) · [TheStreet](https://www.thestreet.com/investing/jensen-huang-issues-blunt-words-on-nvidia-stock)

### 美联邦机构全面清退 Anthropic：国务院、财政部、HHS 已终止合同
继五角大楼将 Anthropic 列为供应链风险后，国务院、财政部、卫生部（HHS）、联邦住房金融局（FHFA）已全面终止与 Anthropic 的合同。Claude 在联邦系统中正被 GPT-4.1、Codex 和 Gemini 替换。
**→ 联邦清退范围比想象中大。但 The Atlantic 同日分析指出：Anthropic 放弃 $2 亿军事合同反而赢得了消费者市场——Claude 下载量日破百万。"失去政府，赢得人心"可能是更值钱的交易。**
📎 [The Atlantic](https://www.theatlantic.com/ideas/2026/03/anthropic-pentagon-contract-openai/686285/) · [NYT](https://www.nytimes.com/2026/03/07/technology/openai-anthropic-pentagon-rivalry.html)

### Chamath：AI token 成本三个月翻三倍，正在把公司从 Cursor 迁走
Chamath Palihapitiya 在 All-In Podcast 上透露：他旗下公司 8090 自 2025 年 11 月以来，AI 推理成本翻了 3 倍多——包括 AWS 推理、Cursor 订阅、Anthropic API。正在考虑将开发工具从 Cursor 迁走以控制成本。
**→ Vibe coding 的隐性成本开始浮出水面。当 AI agent 不只是辅助而是驱动整个 DevOps（见 Cursor Automations），token 消耗会指数增长。这是行业级问题。**
📎 [Business Insider](https://www.businessinsider.com/chamath-palihapitiya-ai-costs-tokens-8090-2026-3)
