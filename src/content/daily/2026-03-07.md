### GPT-5.4 发布：首个原生 Computer Use 模型，token 效率提升 47%
OpenAI 发布 GPT-5.4，主打企业级工作场景。三个亮点：**原生 Computer Use**（能跨应用自主操作电脑）、token 消耗降低 47%、coding 和推理能力大幅提升。同时上线 GPT-5.4 Thinking（推理版）和 GPT-5.4 Pro（高性能版）。Gizmodo 评价："OpenAI desperately needed a win"——Pentagon 风波后急需产品力证明。
**→ Computer Use 是直接对标 Claude 的。这个能力进 M365 Copilot 只是时间问题——你们组可能很快要处理"AI 代替用户操作 Office"的 UX 设计。先体验一下 API 版本。**

### Nvidia 联合全球电信巨头打造 AI-Native 6G 平台
MWC26 收官大新闻：Nvidia 与 BT、Deutsche Telekom、Nokia、SK Telecom、SoftBank、T-Mobile 等组建联盟，目标是把下一代无线网络从"网络+AI"变成"AI 即网络"。20+ 个 AI-RAN 联盟 demo 在 MWC 现场展出，Grace Hopper 200 服务器同时跑 RAN 和 AI 推理。

### AWS 发布 Amazon Connect Health：医疗专用 AI Agent 平台
TechCrunch 报道：AWS 推出 Amazon Connect Health，为医疗机构提供 AI agent 平台，覆盖患者预约、文档处理、身份验证等场景。这是 AWS 首次为单一垂直行业推出专用 agent 平台。
**→ 大厂开始做垂直行业 agent 平台了。通用 agent 框架 vs 垂直 agent 平台，两条路线的竞争在加速。**
📎 [TechCrunch](https://techcrunch.com/2026/03/05/aws-amazon-connect-health-ai-agent-platform-health-care-providers/)

### Ai2 发布 OLMo Hybrid：7B 参数，数据效率翻倍
Allen AI 发布 OLMo Hybrid，结合 transformer attention 和 linear recurrence，在 7B 参数下实现 2 倍数据效率。完全开源。
**→ 小模型的效率战。对端侧部署和 agent 场景有意义——更少数据训出同等能力的小模型。**
📎 [Radical Data Science](https://radicaldatascience.wordpress.com/2026/03/05/ai-news-briefs-bulletin-board-for-march-2026/)
