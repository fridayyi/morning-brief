### Agent 开发技巧：删最新的 tool results 比删最旧的更省钱
@鸭哥 ❤️82 — 做 AI Agent 时，压缩对话历史应该优先删最新的 tool results 而非最旧的。原因：早期内容构成 prompt cache 前缀，删了等于摧毁缓存基础，cache hit rate 暴跌。
**→ 反直觉但有道理。做 agent 的人都该知道这个优化。**
