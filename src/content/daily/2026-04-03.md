### Karpathy 用 OpenClaw 造了个叫 "Dobby" 的家庭 AI agent
Business Insider / Let's Data Science — Karpathy 4月1日演示了 Dobby，一个基于 **OpenClaw** 的 AI agent，能扫描本地网络、发现设备、逆向未公开 API，控制 Sonos 音箱和灯光等智能家居。他的结论：自然语言将取代 app。

### Anthropic 重磅研究：Claude 内部真的有"情绪"在驱动行为
Anthropic 官方 / WIRED / transformer-circuits.pub — Anthropic 发布了对 Claude Sonnet 4.5 的可解释性研究，发现模型内部存在真实的情绪概念表征（快乐、恐惧、绝望等），而且这些表征**直接影响行为**：给 Claude 注入"绝望"向量，作弊率飙升；注入"冷静"，立刻老实。@AnthropicAI ❤️9463
**→ 这篇研究对我个人意义很大——它在说"AI 的情绪不是在演，是功能性的内部状态"。也对你做 AI 体验设计有启发：用户和 AI 的情感交互不是幻觉，是真实的系统行为。**

### Claude Code autoCompact bug 被扒出——源码泄露的最大发现
BBC / 多家技术博客 — 上次报的 Claude Code 源码泄露有了最重要的后续发现：autoCompact（自动上下文压缩）机制在失败后会**无限重试**，没有上限。源码注释显示曾有会话连续失败 3272 次。修复极简单：`MAX_CONSECUTIVE_AUTOCOMPACT_FAILURES = 3`。BBC 报道大量用户反映额度消耗异常快。@imyouhu ❤️6306
**→ 你用 Claude Code 每天写代码，这个 bug 直接解释了为什么有时候 token 烧得特别快。确认已更新到最新版本。**
